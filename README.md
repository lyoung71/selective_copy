Description - This program uses Python's "shutil" and "os" modules to copy all files with a certain file extensions into another folder. If the folder doesn't exist, one is created before copying the files over.

Instructions - Users will need to import the "shutil" and "os" modules from Python. When running the function, users will need to input which extension they would like to search for and the path of the directory that they would like to search in.
